## 第1章 电路的基本概念与基本定律

### 1.1 电路与电路模型

- **电路**：由电气设备和元件按一定方式连接而成的电流通路
- **电路模型**：用理想元件（R、L、C、电源等）近似表示实际电路，便于分析
- 电路的三个组成部分：**电源**（提供电能）、**负载**（消耗电能）、**中间环节**（导线、开关等）
- **集总参数电路**：元件尺寸远小于工作波长，电磁波传播时间可以忽略

### 1.2 电路的基本物理量

#### 电流
- 定义：单位时间内通过导体截面的电荷量，$i = \frac{dq}{dt}$
- 单位：安培（A），1A = 1000mA
- **参考方向**：人为设定的正方向，可任意
- 实际方向与参考方向一致 → 电流为正；反之 → 电流为负

#### 电压
- 定义：电场力把单位正电荷从 a 移到 b 做的功，$u_{ab} = \frac{dW}{dq}$
- 单位：伏特（V）
- 电压方向规定为**高电位→低电位**
- **参考极性**：人为设定，可任意

#### 电位
- 定义：电路中某点对参考点（零电位点）的电压
- 参考点一般选电路中的接地点（地），电位为 0V
- 电位是**相对值**，电压是**绝对值**（两点电位之差）
- **等电位点**：导线直接连接的两点电位相同

#### 电动势
- 定义：在电源内部，非电场力把单位正电荷从负极移到正极做的功
- 方向：在电源内部由低电位指向高电位（从"−"到"+"）

#### 电功率
- 定义：单位时间内电场力做的功，$p = u \times i$
- 单位：瓦特（W）
- **关联参考方向**（电流从电压"+"端流入）：P > 0 → 吸收功率（负载）；P < 0 → 发出功率（电源）
- **非关联参考方向**（电流从电压"+"端流出）：P > 0 → 发出功率（电源）；P < 0 → 吸收功率（负载）
- **功率平衡**：电路中各元件吸收的功率之和 = 发出的功率之和

### 1.3 电阻元件

- **线性电阻**：伏安特性为过原点的直线，满足欧姆定律 $u = Ri$
- **非线性电阻**：伏安特性不是直线（如二极管）
- 电阻单位：欧姆（Ω）
- **电导**：$G = \frac{1}{R}$，单位西门子（S）
- 电阻吸收的功率：$P = i^2R = \frac{u^2}{R}$

### 1.4 电容元件

- 定义：$q = Cu$（电荷 = 电容 × 电压）
- 伏安关系：$i = C\frac{du}{dt}$（电流与电压变化率成正比）
- 电容电压不能突变：$u_C(0^+) = u_C(0^-)$
- 单位：法拉（F），常用 μF、pF
- 储能：$W_C = \frac{1}{2}Cu^2$
- 直流稳态时，电容相当于**开路**（$i = 0$）

### 1.5 电感元件

- 定义：$\Psi = Li$（磁链 = 电感 × 电流）
- 伏安关系：$u = L\frac{di}{dt}$（电压与电流变化率成正比）
- 电感电流不能突变：$i_L(0^+) = i_L(0^-)$
- 单位：亨利（H），常用 mH、μH
- 储能：$W_L = \frac{1}{2}Li^2$
- 直流稳态时，电感相当于**短路**（$u = 0$）

### 1.6 电源元件

#### 理想电压源
- 端电压恒定，与输出电流无关
- 内阻为 0
- 不允许短路（电流会无穷大）
- 符号：圆形中加 "U_s" 或电池符号

#### 理想电流源
- 输出电流恒定，与端电压无关
- 内阻为无穷大
- 不允许开路（电压会无穷大）
- 符号：圆形中加 "I_s" 或箭头

#### 受控源（非独立源）
- 输出电压或电流受电路中其他电压或电流控制
- 四种类型：VCVS（电压控制电压源）、VCCS、CCVS、CCCS
- 受控源在叠加定理和戴维南定理中**不能独立置零**

### 1.7 欧姆定律

- **一段电路**：$U = IR$
- **全电路**：$I = \frac{E}{R+r}$（E 为电动势，r 为内阻）
- 电压源外特性：$U = E - Ir$

### 1.8 基尔霍夫定律 🔥

#### 基尔霍夫电流定律 (KCL)
- $\sum I_{入} = \sum I_{出}$ 或 $\sum I = 0$（流入为正，流出为负）
- **本质**：电荷守恒，节点处不积累电荷
- **推广**：对任一闭合面也成立（广义节点）

#### 基尔霍夫电压定律 (KVL)
- $\sum U = 0$（任一闭合回路，各段电压代数和为零）
- 绕行方向任意选，电压参考方向与绕行方向一致取正，相反取负
- **本质**：能量守恒，沿闭合回路电场力做功为零

### 1.9 电位的计算 🔥🔥

**步骤**：
1. 选定参考点（地），该点 V = 0
2. 确定各支路电流方向
3. 从参考点出发"走"到所求点
4. 经过电阻：顺电流走电位**降低**（−IR）；逆电流走电位**升高**（+IR）
5. 经过电压源：从"+"到"−"降低 U；从"−"到"+"升高 U
6. 遇到接地符号：该点就是参考点，V = 0

**关键结论**：
- 电位值与参考点选择有关
- 电压（电位差）与参考点选择无关
- 等电位点之间可以用导线短接而不影响电路工作状态

## 第2章 电路的分析方法

### 2.1 电阻的串联与并联

#### 电阻串联
- $R_{eq} = R_1 + R_2 + \cdots + R_n$
- 各电阻流过**同一电流**
- **分压公式**：$U_k = U \times \frac{R_k}{\sum R}$

#### 电阻并联
- $\frac{1}{R_{eq}} = \frac{1}{R_1} + \frac{1}{R_2} + \cdots + \frac{1}{R_n}$
- 各电阻两端**电压相同**
- **分流公式**：$I_k = I \times \frac{G_k}{\sum G}$（$G = 1/R$ 为电导）
- 两个并联：$R_{eq} = \frac{R_1 \times R_2}{R_1 + R_2}$

#### 电阻混联
- 先用串并联化简，逐步求出等效电阻

### 2.2 电阻的星形(Y)与三角形(Δ)联结及其等效变换

#### 应用场景
- 在复杂电路中，Y 或 Δ 结构不能直接用串并联化简
- 需要 Y-Δ 等效变换后才能进一步计算

#### Y → Δ 公式
$$R_{ab} = \frac{R_a R_b + R_b R_c + R_c R_a}{R_c}$$
$$R_{bc} = \frac{R_a R_b + R_b R_c + R_c R_a}{R_a}$$
$$R_{ca} = \frac{R_a R_b + R_b R_c + R_c R_a}{R_b}$$

#### Δ → Y 公式
$$R_a = \frac{R_{ab} \times R_{ca}}{R_{ab} + R_{bc} + R_{ca}}$$
$$R_b = \frac{R_{ab} \times R_{bc}}{R_{ab} + R_{bc} + R_{ca}}$$
$$R_c = \frac{R_{bc} \times R_{ca}}{R_{ab} + R_{bc} + R_{ca}}$$

#### 对称情况
- 若三个电阻相等：$R_Y = \frac{1}{3}R_{\Delta}$

### 2.3 电源的两种模型及其等效变换 🔥🔥

#### 电压源模型
- 理想电压源与电阻**串联**
- 伏安关系：$U = U_s - IR$（端电压 = 电动势 − 内阻压降）

#### 电流源模型
- 理想电流源与电阻**并联**
- 伏安关系：$I = I_s - \frac{U}{R}$（输出电流 = 电源电流 − 内阻分流）

#### 等效变换条件
两种模型对外电路等效的充分必要条件：
$$I_s = \frac{U_s}{R} \quad \text{（R 不变，串联变并联）}$$
$$U_s = I_s \times R \quad \text{（R 不变，并联变串联）}$$

#### 注意事项
- 等效是对**外电路**而言，对内分析时不能简单替换
- **理想电压源（R=0）不能等效变换为电流源**
- **理想电流源（R=∞）不能等效变换为电压源**
- 变换时注意电压源正负极与电流源方向的对应关系

### 2.4 支路电流法

以各支路电流为未知量，列写 KCL + KVL 方程组。

- 设 b 个支路，n 个节点 → b 个未知电流
- 列 (n−1) 个 KCL 方程（独立节点数）
- 列 b−(n−1) 个 KVL 方程（独立回路数）
- 共 b 个方程，解 b 个未知量

### 2.5 网孔电流法

- 以网孔电流为未知量
- 每个网孔列一个 KVL 方程
- 方程数 = 网孔数
- 含电流源的支路需特殊处理

### 2.6 叠加定理 🔥🔥🔥

#### 定理表述
在**线性电路**中，任一支路的响应（电压或电流）等于电路中各独立源分别单独作用时在该支路产生的响应的代数和。

#### 求解步骤
1. 依次保留一个独立源，其余电压源短路、电流源开路
2. 分别计算每个独立源单独作用时待求量的分量
3. 将各分量代数相加

#### 重要结论
- **只适用于线性电路**
- 功率不能用叠加定理（$P = I^2R$ 非线性关系）
- 受控源不参与叠加（始终保留在电路中）

### 2.7 戴维南定理 🔥🔥🔥🔥🔥

#### 定理表述
任何一个**线性有源二端网络**，对外部而言，可以用一个**理想电压源 $U_{oc}$ 和电阻 $R_{eq}$ 的串联组合**来等效。

#### 参数求法
| 参数 | 名称 | 求法 |
|---|---|---|
| $U_{oc}$ | 开路电压 | 断开外电路，计算端口电压 |
| $R_{eq}$ | 等效内阻 | ①置零法：内部独立源全置零后端口等效电阻<br>②$U_{oc}/I_{sc}$ 法：$R_{eq} = U_{oc}/I_{sc}$<br>③含受控源时用外加电源法 |
| $I_{sc}$ | 短路电流 | 将端口短路，计算短路电流 |

#### 最大功率传输定理 🔥
- 当 $R_L = R_{eq}$ 时，负载获最大功率
- **$P_{max} = \frac{U_{oc}^2}{4R_{eq}}$**

### 2.8 诺顿定理

- 戴维南定理的对偶形式
- 等效为理想电流源 $I_{sc}$ 与 $R_{eq}$ 并联
- $R_{eq}$ 与戴维南等效电阻相同

### 2.9 最大功率传输定理

- 条件：$R_L = R_{eq}$
- $P_{max} = \frac{U_{oc}^2}{4R_{eq}}$
- 此时效率只有 50%（一半功率消耗在内阻上）

## 第3章 一阶线性电路的时域分析

### 3.1 动态电路的基本概念

- **稳态**：电路中电压电流不随时间变化的稳定状态
- **暂态**：电路从一种稳态过渡到另一种稳态的过程
- **换路**：开关动作、参数突变、电源变化引起的电路状态改变

### 3.2 换路定则 🔥🔥🔥

| 元件 | 定则 | 物理原因 |
|---|---|---|
| 电容 | $u_C(0^+) = u_C(0^-)$ | $W_C = \frac{1}{2}Cu^2$ 不能突变 |
| 电感 | $i_L(0^+) = i_L(0^-)$ | $W_L = \frac{1}{2}Li^2$ 不能突变 |

- **其他量可以突变**：$i_C$、$u_L$、$u_R$、$i_R$ 换路瞬间都可以跳变
- 推广：电容电流、电感电压等不必满足换路定则

### 3.3 初始值的确定

**步骤**：
1. 求 $u_C(0^-)$ 或 $i_L(0^-)$（换路前的稳态值）
2. 由换路定则得 $u_C(0^+) = u_C(0^-)$，$i_L(0^+) = i_L(0^-)$
3. 画出 $t = 0^+$ 时的等效电路：
   - 将电容替换为电压源（值为 $u_C(0^+)$）
   - 将电感替换为电流源（值为 $i_L(0^+)$）
4. 在 $0^+$ 等效电路中求其他量的初始值

### 3.4 稳态值的确定

- $t \to \infty$ 时，电路达到新稳态
- 电容 → **开路**（$i_C = 0$）
- 电感 → **短路**（$u_L = 0$）
- 解直流电路，得到 $u_C(\infty)$、$i_L(\infty)$ 等

### 3.5 时间常数 τ

| 电路类型 | τ | 单位 |
|---|---|---|
| RC 电路 | $\tau = R_{eq}C$ | 秒(s) |
| RL 电路 | $\tau = \frac{L}{R_{eq}}$ | 秒(s) |

- $R_{eq}$：从储能元件（C 或 L）两端看进去的戴维南等效电阻（独立源置零）
- τ 的物理意义：**响应衰减到初始值的 $e^{-1} \approx 36.8\%$（或上升到稳态值的 $63.2\%$）所需时间**
- 工程上认为：$t = 3\tau \sim 5\tau$ 后暂态过程基本结束

### 3.6 一阶电路的零输入响应

**零输入**：没有外加激励，仅靠储能元件初始储能产生的响应。

#### RC 零输入（放电）
- $u_C(t) = u_C(0^+) e^{-t/\tau}$
- $i(t) = \frac{u_C(0^+)}{R} e^{-t/\tau}$
- 电流方向与充电时相反（放电电流）

#### RL 零输入
- $i_L(t) = i_L(0^+) e^{-t/\tau}$
- $u(t) = -Ri_L(0^+) e^{-t/\tau}$

### 3.7 一阶电路的零状态响应

**零状态**：储能元件初始储能为零，仅靠外加激励产生的响应。

#### RC 零状态（充电）
- $u_C(t) = U_s (1 - e^{-t/\tau})$
- $i(t) = \frac{U_s}{R} e^{-t/\tau}$

| t | $u_C$ | 说明 |
|---|---|---|
| 0 | 0 | 初始值 |
| τ | $0.632U_s$ | 上升至 63.2% |
| 3τ | $0.95U_s$ | 基本充满 |
| 5τ | $0.993U_s$ | 工程上认为充满 |

#### RL 零状态
- $i_L(t) = \frac{U_s}{R} (1 - e^{-t/\tau})$

### 3.8 一阶电路的全响应

**全响应** = 零输入响应 + 零状态响应（叠加定理在暂态分析中的体现）

全响应也可以统一用**三要素法**求解。

### 3.9 三要素法 🔥🔥🔥🔥🔥

**通用公式**：
$$f(t) = f(\infty) + [f(0^+) - f(\infty)] e^{-t/\tau}$$

**三要素**：
1. $f(0^+)$：初始值
2. $f(\infty)$：稳态值
3. $\tau$：时间常数

**解题模板**：
1. 求 $u_C(0^+)$ 或 $i_L(0^+)$
2. 求 $u_C(\infty)$ 或 $i_L(\infty)$
3. 求 $\tau = R_{eq}C$ 或 $L/R_{eq}$
4. 代入三要素公式

**适用范围**：
- **一阶线性电路**（只含一个储能元件或可以等效为一个储能元件的电路）
- 不求初值/稳态值时，三要素法也可以直接求解

## 第4章 正弦交流电路

### 4.1 正弦量的基本概念

#### 正弦量的三要素
$$u(t) = U_m \sin(\omega t + \varphi)$$
| 要素 | 符号 | 说明 |
|---|---|---|
| 幅值（最大值） | $U_m$、$I_m$ | 正负峰值 |
| 角频率 | $\omega = 2\pi f = \frac{2\pi}{T}$ | 单位 rad/s |
| 初相位 | $\varphi$ | 与计时起点有关 |

- 工频：$f = 50$Hz，$\omega = 314$ rad/s
- 有效值：$U = \frac{U_m}{\sqrt{2}} \approx 0.707U_m$
- 通常交流电表读数为有效值

### 4.2 同频率正弦量的相位关系

- **相位差**：$\Delta\varphi = \varphi_1 - \varphi_2$
  - $\Delta\varphi > 0$：$u_1$ 超前 $u_2$
  - $\Delta\varphi = 0$：同相
  - $\Delta\varphi = \pm90°$：正交
  - $\Delta\varphi = \pm180°$：反相

### 4.3 正弦量的相量表示法 🔥🔥🔥

#### 为什么要用相量？
- 同频率正弦量的加减可以转化为复数运算
- 微分/积分运算在相量域变成乘除 $j\omega$

#### 相量定义
- 正弦量 $u = \sqrt{2}U\sin(\omega t + \varphi)$ 的相量：$\dot{U} = U\angle\varphi$
- 相量是复数，可以用直角坐标 $a+jb$ 或极坐标 $|\dot{U}|\angle\theta$ 表示
- $j$ 表示 90° 相位旋转（逆时针）
- $-j$ 表示 −90° 相位旋转（顺时针）

#### 相量图
- 用复平面上的矢量表示相量
- KCL、KVL 在相量形式下依然成立

### 4.4 单一参数的正弦交流电路

#### 电阻元件 R
- 相量关系：$\dot{U} = R\dot{I}$
- 电压电流**同相**
- 功率：$P = UI$（只有有功功率），$Q = 0$

#### 电感元件 L
- 感抗：$X_L = \omega L = 2\pi fL$（Ω）
- 相量关系：$\dot{U} = jX_L\dot{I} = j\omega L\dot{I}$
- 电压**超前**电流 90°
- 功率：$P = 0$，$Q_L = I^2X_L > 0$（感性无功为正）
- 直流时 $f = 0$，$X_L = 0$（电感短路）

#### 电容元件 C
- 容抗：$X_C = \frac{1}{\omega C} = \frac{1}{2\pi fC}$（Ω）
- 相量关系：$\dot{U} = -jX_C\dot{I} = \frac{1}{j\omega C}\dot{I}$
- 电压**滞后**电流 90°
- 功率：$P = 0$，$Q_C = -I^2X_C < 0$（容性无功为负）
- 直流时 $f = 0$，$X_C = \infty$（电容开路）

### 4.5 RLC 串联交流电路 🔥🔥🔥

#### 阻抗
$$Z = R + j(X_L - X_C) = R + jX = |Z|\angle\varphi$$
- $|Z| = \sqrt{R^2 + X^2}$（阻抗模，Ω）
- $\varphi = \arctan\frac{X}{R}$（阻抗角）

#### 欧姆定律的相量形式
$$\dot{U} = Z\dot{I} \quad \text{或} \quad \dot{I} = \frac{\dot{U}}{Z}$$

#### 电路性质的判断
| 条件 | 电路性质 | 相位关系 |
|---|---|---|
| $X_L > X_C$，$X > 0$ | 感性 | 电流**滞后**电压（$\varphi > 0$）|
| $X_L < X_C$，$X < 0$ | 容性 | 电流**超前**电压（$\varphi < 0$）|
| $X_L = X_C$，$X = 0$ | 纯电阻（谐振）| 电流电压**同相** |

#### 电压三角形
- $U_R = IR$（与电流同相）
- $U_L = IX_L$（超前电流 90°）
- $U_C = IX_C$（滞后电流 90°）
- $U = \sqrt{U_R^2 + (U_L - U_C)^2}$

### 4.6 阻抗的串联与并联

#### 串联
- $Z_{eq} = Z_1 + Z_2 + \cdots + Z_n$

#### 并联
- 引入导纳 $Y = \frac{1}{Z} = G + jB$
- $Y_{eq} = Y_1 + Y_2 + \cdots + Y_n$

**两个阻抗并联的特殊公式**：
$$Z_{eq} = \frac{Z_1 Z_2}{Z_1 + Z_2}$$

导纳中的术语：
- $G$（电导）：$G = \frac{R}{R^2+X^2}$
- $B$（电纳）：$B = \frac{-X}{R^2+X^2}$
- $B > 0$ 为容性电纳，$B < 0$ 为感性电纳

### 4.7 正弦交流电路的功率 🔥🔥🔥

#### 瞬时功率
$$p(t) = u(t) \times i(t) = UI\cos\varphi[1-\cos 2\omega t] + UI\sin\varphi \cdot \sin 2\omega t$$

#### 有功功率（平均功率）
$$P = UI\cos\varphi = I^2R$$
- 单位：瓦特（W）
- 电阻消耗的电功率，做有用功的部分
- **有功功率守恒**：电路中总有功功率代数和为 0

#### 无功功率
$$Q = UI\sin\varphi = I^2X = I^2(X_L - X_C)$$
- 单位：乏（var）
- 在电源和储能元件（L、C）之间交换的功率
- $Q_L > 0$（感性），$Q_C < 0$（容性）
- **无功功率守恒**

#### 视在功率
$$S = UI = \sqrt{P^2 + Q^2}$$
- 单位：伏安（VA）
- 电力设备容量的额定值
- **视在功率不守恒！**

#### 功率三角形
- 三个功率构成直角三角形：$S^2 = P^2 + Q^2$
- **功率因数**：$\cos\varphi = \frac{P}{S}$

#### 功率因数的提高
- 负载端并联电容器补偿无功
- 补偿容量：$Q_C = P(\tan\varphi_1 - \tan\varphi_2)$
- 并联电容值：$C = \frac{Q_C}{\omega U^2}$

### 4.8 谐振电路 🔥🔥🔥🔥🔥

#### 谐振的概念
- 当电路中 $X_L = X_C$ 时，电压与电流同相，电路处于谐振状态
- 谐振条件：$\omega L = \frac{1}{\omega C}$

#### 谐振频率
$$f_0 = \frac{1}{2\pi\sqrt{LC}}$$
- 只与 L 和 C 有关，与 R 无关
- 改变 L 或 C 可以调节谐振频率

#### 串联谐振（电压谐振）🔥🔥🔥

| 特性 | 谐振时 |
|---|---|
| 阻抗 $Z_0$ | $Z_0 = R$（**最小**）|
| 电流 $I_0$ | $I_0 = U/R$（**最大**）|
| 电感电压 $U_L$ | $U_L = QU$（Q 倍电源电压）|
| 电容电压 $U_C$ | $U_C = QU$（Q 倍电源电压）|

**品质因数**：
$$Q = \frac{\omega_0 L}{R} = \frac{1}{\omega_0 CR} = \frac{1}{R}\sqrt{\frac{L}{C}}$$
- Q 越大，选择性越好，通频带越窄
- **通频带**：$BW = \frac{f_0}{Q}$

#### 并联谐振（电流谐振）🔥🔥🔥

| 特性 | 谐振时 |
|---|---|
| 阻抗 $Z_0$ | 最大 |
| 电流 $I_0$ | 最小 |
| 支路电流 | $I_L \approx I_C = QI_0$（Q 倍总电流）|

### 4.9 频率特性

- **幅频特性**：响应幅值随频率变化的关系
- **相频特性**：响应相位随频率变化的关系
- 串联谐振电路是一个带通滤波器
- Q 值越高，选择性越好，通频带越窄

## 第7章 常用半导体器件

### 7.1 半导体基础知识

#### 本征半导体
- 纯净的硅(Si)或锗(Ge)单晶体
- 共价键结构，每个原子与周围 4 个原子形成共价键
- 本征激发产生自由电子-空穴对
- 温度升高 → 载流子浓度增大 → 导电性增强（负温度系数）

#### 杂质半导体
| 类型 | 掺杂元素 | 多子 | 少子 |
|---|---|---|---|
| **N 型** | 5 价（磷 P、砷 As）| 自由电子 | 空穴 |
| **P 型** | 3 价（硼 B、铝 Al）| 空穴 | 自由电子 |

- 多数载流子浓度 ≈ 掺杂浓度
- 少数载流子浓度与温度密切相关

### 7.2 PN 结 🔥🔥

#### PN 结的形成
- P 型和 N 型半导体接触时，载流子扩散运动
- 交界面形成**空间电荷区**（耗尽层）
- 空间电荷区产生**内建电场**（从 N 指向 P）
- 内建电场阻止进一步扩散 → 达到动态平衡

#### PN 结的单向导电性
| 偏置 | 耗尽层 | 电流 | 电阻 |
|---|---|---|---|
| **正向偏置**（P 接正，N 接负）| 变薄 | 大（mA 级）| 小 |
| **反向偏置**（P 接负，N 接正）| 变厚 | 极小（μA 级）| 很大 |

- 正向导通电压：硅管约 0.7V，锗管约 0.3V
- 反向饱和电流 $I_S$：温度每升高 10°C 约增加一倍

#### PN 结的击穿
- **雪崩击穿**：高反向电压使载流子雪崩倍增
- **齐纳击穿**（隧道击穿）：重掺杂 PN 结在较低电压下的击穿
- 温度系数：雪崩击穿为正温度系数，齐纳击穿为负温度系数

### 7.3 半导体二极管

#### 结构与符号
- PN 结 + 引线 + 管壳
- 阳极（正极）= P 区，阴极（负极）= N 区

#### 伏安特性曲线
| 区域 | 电压范围 | 电流特征 |
|---|---|---|
| 死区 | $0 < U < U_{th}$ | 几乎为 0（硅 $U_{th} \approx 0.5$V）|
| 正向导通 | $U > U_{th}$ | 指数增长，$i = I_S(e^{u/U_T}-1)$ |
| 反向截止 | $-U_{BR} < U < 0$ | $i \approx -I_S$（nA~μA 量级）|
| 反向击穿 | $U < -U_{BR}$ | 电流急剧增大 |

- $U_T = kT/q \approx 26$mV（室温）
- 硅管导通压降 ≈ 0.6~0.7V
- 锗管导通压降 ≈ 0.2~0.3V

#### 主要参数
| 参数 | 符号 | 说明 |
|---|---|---|
| 最大整流电流 | $I_F$ | 长时间工作允许的最大正向平均电流 |
| 最高反向工作电压 | $U_{RM}$ | 允许的最大反向电压（通常为 $U_{BR}$ 的 1/2）|
| 反向电流 | $I_R$ | 反向偏置时的漏电流 |
| 最高工作频率 | $f_M$ | 受 PN 结电容限制 |

#### 二极管电路分析
- **理想模型**：导通时 $U_D = 0V$，截止时 $I_D = 0$
- **恒压降模型**：导通时 $U_D = U_{on}$（硅 0.7V），截止时 $I_D = 0$
- 分析方法：先假设二极管导通/截止，再验证是否符合偏置条件

### 7.4 稳压二极管 🔥🔥🔥🔥🔥

#### 基本特性
- 工作在**反向击穿区**
- 外加反向电压，且 $|U| \geq U_Z$
- 电流在一定范围内变化时，电压基本稳定在 $U_Z$

#### 伏安特性
- 正向特性与普通二极管相同
- 反向电压达到 $U_Z$ 时发生击穿
- 击穿后曲线很陡（$r_Z$ 很小），电压稳定

#### 主要参数
| 参数 | 符号 | 说明 |
|---|---|---|
| 稳定电压 | $U_Z$ | 击穿后的端电压 |
| 稳定电流 | $I_Z$ | 正常工作参考电流 |
| 最小稳定电流 | $I_{Zmin}$ | 维持稳压的最小电流 |
| 最大耗散功率 | $P_{ZM}$ | $I_{Zmax} \times U_Z$ |
| 动态电阻 | $r_Z = \Delta U_Z/\Delta I_Z$ | 越小稳压性能越好 |
| 温度系数 | $\alpha_{UZ}$ | 温度变化 1°C 时 $U_Z$ 的相对变化 |

#### 稳压电路分析

基本电路：输入 $U_i$ → 限流电阻 $R$ → 稳压管 $D_Z$（反向）∥ 负载 $R_L$

**分析步骤**：
1. 检查 $U_i$ 是否足够大（$U_i > U_Z$），确保进入击穿区
2. $U_O = U_Z$（当稳压管正常工作）
3. $I_R = \frac{U_i - U_Z}{R}$
4. $I_L = \frac{U_Z}{R_L}$
5. $I_Z = I_R - I_L$
6. 校验：$I_{Zmin} \leq I_Z \leq I_{Zmax}$

#### 限流电阻的选择
$$R_{min} = \frac{U_{i\_max} - U_Z}{I_{Zmax} + I_{L\_min}}$$
$$R_{max} = \frac{U_{i\_min} - U_Z}{I_{Zmin} + I_{L\_max}}$$

选 $R_{min} < R < R_{max}$

### 7.5 整流与滤波电路

#### 半波整流
- 一个二极管
- **$U_{O(DC)} = 0.45U_2$**（$U_2$ 为变压器次级有效值）
- 纹波大，效率低

#### 全波整流（中心抽头）
- 两个二极管
- **$U_{O(DC)} = 0.9U_2$**
- 需要中心抽头变压器

#### 桥式整流 🔥
- 四个二极管组成二极管桥
- **$U_{O(DC)} = 0.9U_2$**

#### 电容滤波
- 整流输出并联大电容，利用电容充放电平滑输出
- 空载时：$U_{O(DC)} \approx \sqrt{2}U_2 \approx 1.414U_2$
- 有载时：$U_{O(DC)} \approx 1.2U_2$
- **滤波电容越大，纹波越小**

### 7.6 晶体管（双极型晶体管，BJT）

#### 结构
- NPN 型：N 型发射极 + P 型基极 + N 型集电极
- PNP 型：P 型发射极 + N 型基极 + P 型集电极
- 发射区重掺杂，基区薄且轻掺杂，集电区面积大

#### 放大条件
| 结 | NPN 放大状态 | PNP 放大状态 |
|---|---|---|
| 发射结 | 正偏 | 反偏 |
| 集电结 | 反偏 | 正偏 |

#### 三个工作区

| 工作区 | 发射结 | 集电结 | $I_C$ 与 $I_B$ 关系 | 特征 |
|---|---|---|---|---|
| **放大区** | 正偏 | 反偏 | $I_C = \beta I_B$ | 线性放大 |
| 饱和区 | 正偏 | 正偏 | $I_C < \beta I_B$ | $V_{CE} \approx 0.2$V |
| 截止区 | 反偏 | 反偏 | $I_C \approx 0$（$I_{CEO}$ 漏电流）| 关断状态 |

#### 三种基本组态
| 组态 | 输入 | 输出 | 电压增益 | 电流增益 | 输入阻抗 | 输出阻抗 |
|---|---|---|---|---|---|---|
| 共射(CE) | B | C | 大（反相）| $\beta$ | 中 | 中 |
| 共集(CC/射随) | B | E | $\approx$1（同相）| $\beta+1$ | 大 | 小 |
| 共基(CB) | E | C | 大（同相）| $\approx$1 | 小 | 大 |

### 7.7 场效应管 (FET)

#### 与 BJT 的比较
| | BJT | FET |
|---|---|---|
| 控制方式 | 电流控制（$I_B$）| **电压控制**（$U_{GS}$）|
| 输入阻抗 | 中（kΩ 量级）| **极高**（$10^9 \sim 10^{14}\Omega$）|
| 噪声 | 较大 | 较小 |

#### 分类
- **JFET**（结型）：N 沟道 / P 沟道
- **MOSFET**（绝缘栅型）：
  - N 沟道增强型（$U_{GS(th)} > 0$ 才导通）
  - N 沟道耗尽型（零偏压下也能导通）
  - P 沟道增强型（$U_{GS(th)} < 0$ 才导通）
  - P 沟道耗尽型
