# bt's Blog

冰涛的技术博客。基于 [Astro](https://astro.build) 构建，[Astro Cactus](https://github.com/chrismwilliams/astro-theme-cactus) 主题魔改。

## 技术栈

- **框架**: Astro + Tailwind CSS
- **托管**: Cloudflare Pages (`bingtao.xyz`)
- **设计**: SpaceX 玻璃拟态风格

## 本地开发

```bash
pnpm install
pnpm dev        # http://localhost:4321
pnpm build      # 构建到 dist/
```

## 部署

推送 `main` 分支到 GitHub，Cloudflare Pages 自动构建部署。
